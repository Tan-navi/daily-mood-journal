# 🌿 Daily Mood Journal

A simple and interactive JavaScript web application that allows users to record their daily mood and write a short journal entry.

## ✨ Features

- Enter your name
- Select your current mood
- Write a journal entry
- Form validation
- Dynamic result display
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Daily-Mood-Journal/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your web browser.

## 📸 Project Preview

The application lets users save a mood entry by entering their name, selecting a mood, and writing a short journal note. The entered information is displayed instantly on the page using JavaScript.
